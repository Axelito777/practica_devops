# Practica DevOps

Repositorio de practica para el curso ISY1101.

## Estrategia de ramas
Se utiliza Feature Branches (GitFlow):
- feature/frontend: contiene la pagina web
- feature/docker: contiene el Dockerfile y docker-compose
- feature/docs: contiene la documentacion del proyecto

## Tecnologias
- HTML
- Docker + nginx
- AWS EC2
- GitHub Actions
